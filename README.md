# Brick Next Pipes

## Usage

```shell
npm install --save @easyops-cn/brick-next-pipes
```

```ts
import { pipes } from "@easyops-cn/brick-next-pipes";

const stringified = pipes.yamlStringify(yourObject);
```

## Development

```shell
npm install
npm start
```

## Testing

```shell
npm test
```

## Publish

```shell
npm run build
npm publish --access public
```
